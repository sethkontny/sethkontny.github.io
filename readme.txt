1. Added toggle menu button for small screens
2. Added another version of services section (without circles)
3. Added 404 page
4. Added disable option for landing section animation and logo animation
   - To disable landing section animation, add class "nomove" to intro section:
     <div id="intro" class="nomove">
   - To disable logo animation, add class "nospin" in span tag of the logo:
     <div id="logo"><a href="#"><span class="nospin">Typebig</span></a></div>
